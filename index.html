Keys-To-The-Tunnel

use:

    ssh -T -R PORT-FROM-ABOVE:127.0.0.1:PORT-ON-DEV GH-HANDLE@medic-tunnel.plip.com

example:

    expose local port 80 on https://yrimal-ssl.medic-tunnel.plip.com  using yrimal's GH keys

    ssh -T -R 3089:127.0.0.1:80 yrimal@medic-tunnel.plip.com

note:

    Caddy is configured on the http hosts to speak http to your localhost
    server. Conversely, it is configured to speak https on the https hosts.
    If you mix these up it will try and speak https to http (or vise versa)
    and it will fail.
